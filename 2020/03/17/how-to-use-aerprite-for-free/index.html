Aseprite 是目前较为主流的像素画软件，并且十分良心的提供了源码.
于是我们可以通过自己编译源代码来获得一份自用的可执行版本。
本篇文章将主要讲解Mac下的编译过程。
如果您觉得麻烦或是有钱不在乎，可以前往 Steam 或者 Aseprite.org 自行购买（前者 48 元，后者 15 美金）

首先感谢 Aseprite官方文档 的详细指导。

需要的东西

  • 最新版的CMake（≥3.14)
  • Ninja
  • 一份编译版本的 aseprite-m81
  • macOS10.15 SDK
  • Xcode 11.2.1

获取源代码

您可以从这里获取Aseprite-v1.x-Source.zip 最新的 Aseprite 版本下载文件（在这种情况下，请按照.zip文件内的编译说明进行操作）：
https://github.com/aseprite/aseprite/releases

安装 CMake

在官网可以轻松找到链接 CMake下载
你会得到一个 .dmg 文件，先安装它并打开，在菜单栏里的 Tools/How to Install For Command Line Use 可以知道如何安装到终端，跟着做就行。

安装Ninja

你可以在这里找到下载：Ninja

一份已编译版本的aseprite-m81

你可以在这里找到下载： Aerprite
